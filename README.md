### Basic Usage

### Variables
```css
/**************************************\
               bootstrap
\**************************************/
@import url("plugins/bootstrap.min.css");

                selects
\**************************************/
@import url("plugins/flatpickr.css");
@import url("plugins/nice-select2.css");
```
